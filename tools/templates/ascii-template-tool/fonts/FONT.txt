Boothclub Mono (identity font)

Canvas / @font-face
  BoothclubMono.woff2  — optional (browser prefers it if present)
  BoothclubMono.woff   — fallback

Outline SVG export (opentype.js)
  fonts/boothclub-outline-embed.js — REQUIRED for reliable export. It inlines
  the WOFF as base64 so export works when you open index.html as file://
  (browsers block fetch() to local font files in that mode).

  Regenerate after you change the font file:
    python3 -c "
    import base64, pathlib
    p = pathlib.Path('BoothclubMono.woff')
    b64 = base64.b64encode(p.read_bytes()).decode('ascii')
    pathlib.Path('boothclub-outline-embed.js').write_text(
      \"window.__BOOTHCLUB_WOFF_B64='\" + b64 + \"';\\n\", encoding='ascii')
    "
  (run from this fonts/ directory)

  If the embed script is missing, export tries fetch(fonts/BoothclubMono.woff)
  instead — that only works when the tool is served over http(s), not file://.

CDN scripts (pako + opentype.js) must load for outline export; use a network
or vendor them locally if you work fully offline.
